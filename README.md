# SignatureCNNBot

A CNN-based Signature Verification system using a Siamese Neural Network,
deployed as a Telegram Bot.

## Features
- Siamese CNN for signature similarity
- Genuine vs Forged verification
- Best-K aggregation
- PDF report generation
- Telegram bot interface

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- python-telegram-bot
